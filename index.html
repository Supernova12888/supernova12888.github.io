What is JavaScript?

JavaScript is a computer programming language to make stuff. You can use it to make games, apps, and websites (just like this webpage). JavaScript is one of the most important and commonly used programming language. If you want to learn coding, you should learn JavaScript.

What is this webpage about?

This website is about JavaScript. You can learn from basic things (such as variables) to more advanced stuff (like how to use push and splice).

    Variables

    Variables are very important in JavaScript and you'll use them a lot once we get into more advanced JavaScript. So what are variables? Variables are used to replace numbers, strings, and many other things.

    When you create a new variable, you always start with var. The examples above shows several ways to create a variable. There are a few rules to naming a variable.

    • It cannot start with a number.
    • It cannot include any symbols except for the dollar sign or underscore.
    • It cannot be a JavaScript keyword.

    Math

    Here's a list of some math objects:

    • Math.round(2.3) - Rounds 2.3 to the nearest integer (2).
    • Math.floor(2.3) - Rounds 2.3 down to the nearest integer (2).
    • Math.ceil(2.3) - Rounds 2.3 up to the nearest integer (3).
    • Math.pow(4, 2) - Returns the first value (4) to the power of the second value (2). (Returns 16.)
    • Math.sqrt(36) - Returns the square root of a number. (Returns 6.)
    • Math.abs(-2.3) - Returns the absolute value of -2.3 (2.3).
    • Math.sin(30) - Returns the sine of a number (in radians).
    • Math.cos(30) - Returns the cosine of a number (in radians).
    • Math.tan(30) - Returns the tangent of a number (in radians).
    • Math.asin(30) - Returns the inverse sine of a number (in radians).
    • Math.acos(30) - Returns the inverse cosine of a number (in radians).
    • Math.atan(30) - Returns the inverse tangent of a number (in radians).
    • Math.min(2, -5, 7) - Returns the smallest number (-5).
    • Math.max(3, 7, 5) - Returns the largest number (7).
    • Math.random() - Returns a random number between 0 and 1.
    • Math.PI - Returns pi (3.141592653589793).

    Push and Splice

    You can use push() and splice() to add or remove something in an array. push() has one argument, and that is what you are adding to the array. splice() has two arguments; the first one is what you are removing and the second is how many you're removing.
